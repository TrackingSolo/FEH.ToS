Section 508 Compliance Statement

Accessibility Commitment

FEH is committed to ensuring digital accessibility for all users, including those with disabilities. We strive to adhere to Section 508 of the Rehabilitation Act and the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Accessibility Features Implemented

1. Screen Reader Compatibility

  • Semantic HTML: All content uses proper heading structures and landmarks
  • Alt Text: Descriptive alternative text for all images, including property photos and map elements
  • Form Labels: Clear, descriptive labels for all form inputs
  • Property Listings: Structured data that announces as "3-bedroom house, $250,000, 2.5 miles from Richmond VA Medical Center"
  • Navigation: Logical tab order and skip links for efficient navigation

2. Keyboard Accessibility

  • Full Functionality: All features accessible via keyboard navigation
  • Visible Focus: Clear focus indicators on all interactive elements
  • Tab Navigation: Logical tab order through property listings, filters, and maps
  • Keyboard Shortcuts: Standard shortcuts for common actions
  • Modal Dialogs: Proper focus management in pop-up windows

3. Visual Accessibility

  • Color Contrast: Minimum 4.5:1 contrast ratio for normal text, 3:1 for large text
  • Color Independence: Information not conveyed by color alone
  • Font Scaling: Support for text enlargement up to 200% without horizontal scrolling
  • Responsive Design: Content remains usable at all zoom levels
  • High Contrast Mode: Support for system high contrast settings
